Obtener repositorio remoto:

1. Clonar repositorio
    git clone url (nombre Opcional)
2. Importar ramas
    git checkout rama
3. Importa el repositorio remoto
    git fetch
4. Actualiza el puntero master local a origin/master
    git merge origin/master

3. git pull //realiza los pasos 3 y 4 solos
